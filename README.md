Smart Meet
Smart Meet is a web-based video conferencing application that enables users to connect and communicate in real time. It allows users to create or join meeting rooms and interact through video, audio, and chat directly from the browser.

🚀 Features
🔹 Real-time video & audio communication
🔹 Create and join meeting rooms
🔹 Peer-to-peer connection using WebRTC
🔹 Live chat functionality
🔹 Simple and user-friendly interface

🛠️ Tech Stack
Frontend: React.js
Backend: Node.js, Express
Real-time Communication: WebRTC, Socket.io
Database: MongoDB

How It Works
Users create or join a meeting room
A connection is established using WebRTC
Socket.io handles signaling between users
Media streams are shared in real time.
